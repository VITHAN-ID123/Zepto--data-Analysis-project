# Zepto Data Analysis Project

## Overview
Analyzed Zepto product and pricing data using SQL to identify
high-revenue categories, stock-out issues, and pricing inconsistencies.

## Tools Used
- SQL Server
- Excel (for validation)
- GitHub

## Key Analysis
- Top 10 products by MRP
- Out-of-stock high-demand products
- Category-wise estimated revenue
- Price normalization (paise to rupees)

## Key Insights
- Certain high-MRP products frequently go out of stock
- Data had pricing stored in paise causing incorrect analytics
- Revenue concentration in few categories
